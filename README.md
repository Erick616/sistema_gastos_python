# Sistema de gastos personales en Python

Aplicación de consola desarrollada en Python para registrar ingresos y gastos, calcular el saldo disponible y guardar la información en un archivo JSON.

## Funciones

- Registrar ingresos
- Registrar gastos
- Clasificar gastos por categoría
- Ver todos los movimientos
- Calcular el total de ingresos
- Calcular el total de gastos
- Mostrar el saldo disponible
- Consultar gastos por categoría
- Guardar la información en JSON
- Recuperar los datos al abrir nuevamente el programa

## Tecnologías utilizadas

- Python
- JSON
- Git
- GitHub
- Visual Studio Code

## Cómo ejecutar el programa

1. Clona o descarga este repositorio.
2. Abre la carpeta en Visual Studio Code.
3. Abre una terminal dentro de la carpeta.
4. Ejecuta:

```bash
python main.py
